# Course Tracking Application

A Flask-based web application that allows users to track course availability and seat counts for their selected classes.

## Features

- User authentication (signup/login)
- Course tracking by CRN (Course Reference Number)
- Real-time seat availability updates
- Term-specific course monitoring
- Secure password handling
- SQLite database storage

## Tech Stack

- Backend: Flask, SQLAlchemy
- Frontend: HTML, JavaScript
- Database: SQLite
- Authentication: Flask-Login
